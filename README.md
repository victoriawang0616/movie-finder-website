# movie-finder-website
Movie Finder Website - Agile Web Development Project

CineFind is a movie discovery website. Users can search for any film, view detailed information including ratings, release year, and descriptions, and save their favourites to a personal watchlist.  
From a technical side, we used HTML5 for structure, CSS3 for styling with cinematic animations, and vanilla JavaScript for all the logic. 
We integrated the TMDB API — The Movie Database — to pull in real movie data including posters and ratings.
 For storing user accounts and watchlists, we used the browser's localStorage, which means everything works without a backend server. And throughout the project, we used Git and GitHub for version control and team collaboration
